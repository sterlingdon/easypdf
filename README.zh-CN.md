# Easy PDF

> **不再为 PDF 发愁。**

[English README](./README.md)

Easy PDF 是一个功能强大、全面且设计精美的 Web 应用程序，致力于修改、转换和管理 PDF 文件。我们的使命很简单：提供一套强大的 PDF 工具，其用户界面比市场上的任何其他产品都更洁净、更快速、更直观。

我们相信，强大的功能不应以牺牲易用性为代价。Easy PDF 为您带来专业级的 PDF 操作工具，并将其包裹在极简、易用的设计中。

## 开发初衷

PDF 编辑和操作是一个竞争者非常多的领域，但现有产品大多收费且体验不佳。正所谓 **“天下苦 Adobe 久矣”**。

如今 AI 强大了，制作软件的成本越来越低，是时候改变 PDF 行业了。

## 我们的使命

我们希望世界上的每个人都能使用免费的 PDF 处理软件。为了实现这一目标，Easy PDF 不仅提供 Web 应用程序，还将被打包为支持 **macOS** 和 **Windows** 的客户端软件。

## 功能特性状态

### 🛠️ 组织与编辑
- [x] **合并 PDF**: 将多个文件合并为一个无缝文档。
- [x] **拆分 PDF**: 按页面范围拆分文档。
- [ ] **页面管理**: 重新排序、添加或删除页面。
- [ ] **删除页面**: 从文档中删除特定页面。
- [ ] **按大纲拆分**: 根据书签/大纲拆分 PDF。
- [ ] **按大小拆分**: 将 PDF 拆分为特定大小的文件。
- [ ] **删除空白页**: 自动检测并删除空白页面。
- [ ] **多页拼版**: 将多个页面合并到一张纸上。
- [ ] **裁剪**: 裁剪 PDF 页面。
- [ ] **调整大小**: 更改页面尺寸（调整大小/缩放）。
- [ ] **旋转**: 旋转页面或整个文档。

### 🔄 转换
- [x] **PDF 转图片**: 将 PDF 页面转换为图片。
- [x] **图片转 PDF**: 从图片创建 PDF。
- [ ] **PDF 转 JPG**: 专门转换为 JPG。
- [ ] **PDF 转 PNG**: 专门转换为 PNG。
- [ ] **PDF 转 WebP**: 专门转换为 WebP。
- [ ] **PDF 转 AVIF**: 专门转换为 AVIF。
- [ ] **PDF 转 SVG**: 将 PDF 转换为矢量 SVG。
- [ ] **PDF 转 HTML**: 将 PDF 转换为 HTML。
- [ ] **PDF 转文本**: 提取纯文本。
- [ ] **长图**: 将 PDF 页面拼接成长图。

### 🧩 内容提取与修改
- [ ] **提取页面**: 提取特定页面。
- [ ] **提取图片**: 提取嵌入的图像。
- [ ] **提取路径**: 提取矢量路径 (SVG)。
- [ ] **提取文本**: 提取文本内容。
- [ ] **删除文本**: 从 PDF 中删除文本。
- [ ] **删除图片**: 从 PDF 中删除图像。
- [ ] **删除矢量**: 删除矢量元素。

### 🔒 安全与优化
- [x] **压缩 PDF**: 减小文件大小。
- [ ] **解锁 PDF**: 移除密码。
- [ ] **保护 PDF**: 添加密码。
- [ ] **灰度化**: 转换为灰度/黑白。
- [ ] **查看元数据**: 查看 PDF 元数据。
- [ ] **编辑元数据**: 修改 PDF 元数据。

## 设计理念

Easy PDF 建立在 **“二次创新”** 的原则之上。我们将经过验证的实用程序通过设计进行了提升。
- **极简界面**：没有杂乱，只有您需要的工具。
- **以用户为中心的操作**：简化复杂任务的工作流程。
- **隐私至上**：所有处理都针对安全性和安心进行了优化。
